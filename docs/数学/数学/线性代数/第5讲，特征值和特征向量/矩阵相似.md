# 总览
普通的矩阵相似没有太多价值，只有相似对角化才有价值。
因此，对于矩阵相似这样的题，它最多考察的是**已知两个矩阵,判断两个矩阵是否相似**这样的题目。
其他的题目，很难用普通矩阵相似来考察，因为做不出来啊。
