# 堆的基本概念
1. 堆必须是一个完全二叉树
2. 根据堆自身的顺序性，可以将堆分为大根堆和小根堆
	堆不考虑左右子树的大小关系，只考虑子节点和父节点之间的关系。
 
# 大根堆和小根堆
1. 大根堆——**要求堆的父节点比两个子节点都要大，即最大值就是根节点。
2. 小根堆——要求堆的父节要比两个子节点都要小，即最小值就是根节点。 |
