# 平衡二叉树的插入
平衡二叉树在插入过程中，**会形成一个最小不平衡子树，我们插入在调整的过程中，就是要调整这个最小不平衡子树。（并且只调整这个最小不平衡子树，网上的其他资料都是假的）**

**实质上，我们调整最小不平衡子树，就是要让这个二叉平衡树在调整前和调整后的中序遍历都是一样的。**
