# 完全二叉树节点个数与高度问题
$高度h = L(\log_2^{n}) + 1 (L代表向下取整）$
完全二叉树可以认为是有一部分的满二叉树。


| 完全二叉树    | Complete Tree | 叶子节点只会出现倒数第1层，倒数第1层的叶子节点都靠左对齐              |
| -------- | ------------- | ------------------------------------------ |
| 满二叉树     | Full Tree     | 所有中间结点都有两个孩子。                              |
| 完美二叉树    | Perfect Tree  | 最漂亮的树，每一层的节点都是满的。                          |
| AVL平衡二叉树 | AVL Tree      | 首先是一棵二叉搜索树，其次保证所有的节点（左子树高度-右子树高度）的绝对值 <= 1 |
